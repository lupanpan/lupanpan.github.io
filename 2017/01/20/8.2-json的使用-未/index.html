内容：XML的介绍、json的解析与使用方式、XML与json的区别

1、 XML

可扩展的标记语言，它里面的标签是自定义的，例如

xml与html的区别

  1. xml 可扩展的标记语言
    html 超文本标记语言
  2. xml 标签是自定义的
    html 标签是w3c指定的，需要按照w3c指定的规范去写
  3. xml 主要用来做软件的配置文件、数据传输，一般自己写自己解析
    html 用来做界面显示，浏览器解析

2、 JSON

即 JavaScript Object Notation，另一种轻量级的文本数据交换格式，独立于语言。

客户端跟服务器进行交互的时候，为了性能，交互的次数越少越好，所以我们用到了缓存，交互的时候传递的数据越少越好。

1、数据在名称/值对中
2、数据由逗号分隔(最后一个健/值对不能带逗号)
3、花括号保存对象方括号保存数组
4、使用双引号

JSON数据在不同语言进行传输时，类型为字符串，不同的语言各自也都对应有解析方法，需要解析完成后才能读取

2.1、PHP解析方法
json_encode()、json_decode()

2.2、Javascript 解析方法
JSON对象的转换:

方法 说明
eval(); 字符串转换为json，浏览器提供的方法
JSON.parse(); 字符串转换为json，js的api，有兼容性问题
JSON.stringify()； json转换为字符串

总结：JSON体积小、解析方便且高效，在实际开发成为首选。
