内容：阻止事件冒泡与默认事件、return false的注意事项

js事件绑定方式

  1. 行内绑定
  2. 给DOM元素直接绑定 btn.onclick
  3. addEventListener / attachEvent

阻止冒泡:

event.stopPropagation();

event.cancelBubble = true;

阻止默认事件:

event.preventDefault();

event.returnValue = false;

return false的注意事项

  1. 只能在DOM元素直接绑定事件的这种绑定方式中能阻止默认行为；
  2. 在addEventListener这种绑定事件中不能阻止默认行为；
  3. return false在原生js中不能阻止事件冒泡；
  4. 在jQuery中既可以阻止冒泡，也可以阻止默认行为；
